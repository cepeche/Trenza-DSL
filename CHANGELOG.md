# Changelog

All notable changes to the Trenza DSL project will be documented in this file.

## [v0.0.1] - 2026-03-20
### Cambiado
- Renombramiento integral de Helix a Trenza en toda la especificación y herramientas.
- Repositorio reestructurado separando `spec/`, `history/` y `docs/` para salvaguardar la historia probatoria del co-diseño ("Cuaderno de Laboratorio").
- Los archivos `.helix` cambian de extensión a `.trz`.
- Palabra reservada `bloqueado` cambiada por `forbidden`.
- Licencia cambiada a Dual: AGPL-3.0 con opción a licencia comercial privativa para proveedores de Modelos de IA.

### Nota Histórica
El estado previo al renombramiento y la organización de prototipo (`docs/` plana y `examples/`) se preservan íntegramente en el tag de git `v0.0.0`.
